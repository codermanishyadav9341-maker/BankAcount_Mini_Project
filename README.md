Project Description

This is a Java based Bank Account Mini Project.
It allows users to create accounts, deposit money, withdraw money and check balance.

🚀 Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Balance
- Simple Console Based Application

🛠️ Technologies Used

- Java
- OOP Concepts
- IntelliJ IDEA

▶️ How to Run

1. Clone the repository
2. Open project in IntelliJ IDEA
3. Run Main.java file
